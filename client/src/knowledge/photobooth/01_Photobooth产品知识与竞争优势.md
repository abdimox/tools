# Photobooth 产品知识与竞争优势

## 一句话定义

Photobooth 不是简单的拍照设备，也不是普通打印机。它是把现场互动、照片拍摄、预览重拍、专属版式、即时打印和实体纪念结合在一起的活动体验区。

## 标准体验流程

1. 宾客来到互动拍照区。
2. 工作人员引导站位和构图。
3. 屏幕显示实时画面。
4. 倒计时完成拍摄。
5. 宾客预览照片，不满意可以重新拍摄。
6. 系统套入活动专属版式。
7. 确认后现场打印。
8. 宾客领取实体照片。
9. 系统保存电子照片，便于后续交付或补打。

实际服务是否全部包含以上环节，必须以真实套餐为准。

## 客户真正购买的东西

客户表面上租的是设备，实际上购买的是：

- 宾客等待期间的互动项目
- 一个容易参与的现场打卡点
- 统一风格的高质量合照
- 可以马上带走的实体纪念
- 婚礼姓名、日期或企业 Logo 的专属视觉
- 主办方后续可以使用的活动照片
- 宾客愿意分享的社交内容

销售和内容表达应优先讲这些结果，不要长期停留在机器外观和功能清单。

## 六项核心优势

### 1. 画面更稳定

固定机位、固定距离、合理补光和经过测试的版式，更容易保持亮度、构图、角度、背景和裁切的一致性。

### 2. 可以预览和重拍

闭眼、表情不满意、人物被裁切、多人没有看镜头等情况，可以在打印前发现并重新拍摄，减少废片。

### 3. 适合多人连续体验

配合适合活动使用的打印设备和现场引导，更适合婚礼、企业活动等宾客较多的场景。实际接待能力需要结合真实设备速度和流程确认。

### 4. 实体照片有礼物感

电子照片容易沉在手机相册里，现场拿到的实体照片更像伴手礼或活动纪念品。

### 5. 视觉可以定制

可以围绕婚礼姓名、日期、企业 Logo、品牌颜色和活动主题设计照片版式，使照片成为活动视觉的一部分。

### 6. 同时解决互动与纪念

普通摄影主要负责记录，Photobooth 还承担迎宾娱乐、员工互动、打卡、礼物和传播素材等功能。

## Photobooth 与即影即有

两者都能在现场获得实体照片，但定位不同。

| 对比 | Photobooth | 即影即有 |
|---|---|---|
| 核心定位 | 完整互动拍照体验 | 即时成像工具 |
| 拍摄方式 | 固定机位，通常有现场引导 | 手持移动拍摄 |
| 拍摄预览 | 支持实时画面 | 通常不支持 |
| 打印前确认 | 可以先确认再打印 | 按下快门即消耗相纸 |
| 重拍 | 不满意可以重拍 | 重拍会再次消耗相纸 |
| 多人构图 | 更容易控制 | 容易受镜头和距离影响 |
| 补光 | 可配置稳定补光 | 更依赖现场和相机自身曝光 |
| 电子文件 | 可以保存 | 传统胶片通常没有 |
| 补打 | 可以从记录中补打 | 通常无法复制原片 |
| 视觉定制 | 版式、Logo、日期、主题 | 定制空间较少 |
| 使用规模 | 更适合中大型活动 | 更适合少量、移动和随机记录 |

### 正确对外表达

> 即影即有适合少量、随机、移动和原生胶片感的记录；Photobooth 更适合宾客较多、需要预览重拍、统一视觉、电子保存和后续补打的活动。

### 不应使用的表达

- Photobooth 完全取代即影即有
- Photobooth 一定比所有即影即有更清晰
- 绝对不会排队
- 无限打印、无限拍摄（除非套餐确实如此）
- 任何未经确认的相机、打印速度和照片数量

## 三个场景的价值翻译

### 婚礼

- 迎宾等待不无聊
- 新人不用逐桌与每位宾客合照
- 宾客带走专属照片
- 长辈、朋友和同事都能参与
- 照片自然带有新人姓名和婚礼日期

### 宝宝宴

- 把难得聚齐的家庭成员留下来
- 长辈操作门槛低
- 亲子和家庭合照可以立即带走
- 宝宝姓名、生日和主题可以进入版式

### 企业活动

- 签到后有事可做，减少冷场
- 员工更容易自然交流
- 企业 Logo 和主题进入照片
- 照片既是员工纪念，也是活动传播内容
- 一项服务同时承担互动、拍摄和纪念品
